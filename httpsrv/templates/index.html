Welcome to SpacEd MANAGEMENT (pronounced: "space ed")

What is this?

SpacEd is a novel spaced learning platform.

For learners, we offer a method of learning that allows you to exercise your long term memory of material you're interested in.

For instructors, we offer an intelligent framework allowing you to create courses composed of question/answer templates, which SpacEd uses to generate specific instances of question/answer pairs.

When a learner joins one of your courses, they'll be asked one instance of each of your course's question templates.

From that point, each question template has a stateful relationship with that learner, and that relationship is defined by two important values: a remaining value and an increment value.

The remaining value (an integer, like 0, 1, or 2), indicates how many {{base_unit}} are left until the learner is asked another instance of this question template. When this number reaches 0, the learner gets a notification that they have questions ready to be answered.

The increment value, also an integer, defines by how much the remaining value is increased whenever the learner answers a question correctly. This value can also be called the re-exposure delay, because it determines what the delay should be between two prompts from the same question template.

The general pattern is, wrong answers trigger a shorter re-exposure delay while correct answers trigger longer re-exposure delays. If you understand something, the system aims to push your memory of that material farther until it identifies a point where you forget the answer.

When an instance of a question template is answered correctly by the learner, the remaining value gets incremented by increment, and increment increases by a value calculated dynamically by SpacEd using data from other learner-template relationships. The goal in this calculation is to identify and converge upon the optimal re-exposure delay for a given question template, and perhaps to derive similar insight about their parent subject areas.
